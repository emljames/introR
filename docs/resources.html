Beyond the workshop

Google is your friend

Want to know how to do something? Google it. There are many options for help out there, including:

  • Help files for particular functions
  • Open source textbooks
  • Tutorials attached to specific packages
  • Bespoke tutorials written by random people
  • Help forums - e.g., StackOverflow

Data processing in R

  • DataCamp - interactive courses for programming in R (free introductory levels, ££ for more advanced)
  • CodeAcademy - interactive exercises for introductory R
  • Data Carpentry - slightly longer workshop materials for similar content as we covered today.
  • Software Carpentry - similar style materials again, but slightly more content, includes further information on writing your own functions and using R Markdown.
  • R for Data Science - textbook written by the author of the tidyverse himself, freely available online
  • Gorilla to Tidy Data - Emma’s tutorial for processing output files from the online experiment platform Gorilla. This is similar in style to the workshop today, but directly applied to processing messy output from Gorilla. Part 2 also extends to some more complex challenges that you might encounter, including reshaping and combining datasets.
  • PsyTeachR- there are plenty of excellent resources that cover different aspects of data processing and undergraduate-level statistics from the psyTeachR team at the University of Glasgow.

Cheat sheets

Cheat sheets are handy summaries of key tools. I used to print these and stick them by my desk for easy reference! You’ll find cheatsheets for dplyr, tidyr, and ggplot2 here.

Advanced Research Methods (Psychology)

If you want to see more of what R can do, you’re welcome to audit the Advanced Research Methods module (taught by both Emma and Jamie). You can be added to the VLE, and/or attend the timetabled sessions on Mondays and Tuesdays.

The remaining topics timetabled for this term are: SEM, Multivariate Pattern Analysis, Fourier Analysis, Bayesian Methods, and Reproducible Research Methods. The main focus of this module is on theoretical application (lectures Mondays 2-3pm), but there is also a practical session on Tuesdays that includes a handout you could work through independently.

Next year, this module will likely run in Semester 1 (R light), with a more advanced version running in Semester 2 (R heavy!).Ask Emma if you would like more information on this.
